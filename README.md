# 北大言語学サークル Huling

## develop

See [Wiki](https://github.com/hulinguistics/huling/wiki).

```bash
git clone -b dev git@github.com:hulinguistics/huling.git
cd huling
yarn
yarn dev
```

## 記事作成

### 言語概説

```bash
yarn util:lang new チベット語
```

### オリンピック問題解説

```bash
yarn util:olym new "JOL2021-1 カビル語"
```
