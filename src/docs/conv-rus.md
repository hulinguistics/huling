軟音記号 ь は `'` / `q`, 硬音記号 ъ は `''` / `qq`,  
ダッシュ — は `--`, 引用符 « » は `" ""`.  
アクセント記号は母音字の直後に `/` または `w` を入れてください (`//` で第2アクセント).
1918年の正書法改革以前に使用されていた文字 ѣ, і, ѵ, ѳ はそれぞれ `e_`, `i_`, `i__`, `f_`.
