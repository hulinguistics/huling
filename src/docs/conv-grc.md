- **無気記号**：語頭のとき自動的に付加, **有気記号**：`h + 母音字`
- **鋭アクセント**`/`, **重アクセント**`//`, **曲アクセント**`///`  
`/` の代わりに `q` も使用可能.
- **短母音**：`a_`, `i_`, `y_`, **長母音**：`aa`, `ee`, `ii`...
- **下書きのイオタ**：`aai`, `eei`, `ooi`, `aa/i` など.
- **シグマの語末形**：`s.`, `s;`, `s + 半角スペース` など.
