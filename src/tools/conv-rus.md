- **軟音記号 ь** : `'` / `q`, **硬音記号 ъ** : `''` / `qq`
- **ダッシュ —** : `--`, **引用符 « »** : `" ""`
- **アクセント記号** : `/` または `w` (`//` で第2アクセント)
- **廃字 ѣ, і, ѵ, ѳ** : `e_`, `i_`, `i__`, `f_`
